Ausführliche Beschreibung

struct to store result of comparison of b-trees

Dokumentation der Aufzählungstypen

anonymous enum

only valid if is_equal == false DEPTH if the trees have a different depth NUMBER_OF_ELEMENTS if a node was found in which the number of elements differs ELEMENT if an element was found of which the value differs

Dokumentation der Datenelemente

int compare_result::depth

only valid if is_equal == false and difference != DEPTH contains the level on which a difference was found. the root node is level 0

enum { ... } compare_result::difference

only valid if is_equal == false DEPTH if the trees have a different depth NUMBER_OF_ELEMENTS if a node was found in which the number of elements differs ELEMENT if an element was found of which the value differs

int compare_result::is_equal

true (1) if the trees are equal, false (0) otherwise

int* compare_result::path

only valid if is_equal == false and difference != DEPTH contains a path from the root node to the node in which the difference was found. so {1} means the first child of the root node and {2, 0} means the zeroth of the second child of the root node. this array contains depth values. if difference == ELEMENT, this contains an additional value for the index of the elemennt.
